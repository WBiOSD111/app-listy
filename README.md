# Listy App
> Showcases the effect of the iPhone's built-in accelerometer.

## Features

- [x] CoreMotion and CMMotionManager
- [x] Receiving accelerometer updates
- [x] Handling accelerometer data
- [x] Writing the parallax function using UIInterpolatingMotionEffect
- [x] Writing the parallax function using the accelerometer

## Build

- iOS 11.0
- Xcode 9.4.1
